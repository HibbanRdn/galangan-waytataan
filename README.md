## ğŸš€ Cara Menjalankan Project

Ikuti langkah-langkah di bawah ini untuk menjalankan project di lingkungan lokal kamu:

### 1. Clone Repository
```bash
git clone [https://github.com/HibbanRdn/galangan-way-tataan.git](https://github.com/HibbanRdn/galangan-way-tataan.git)
cd galangan-way-tataan

```

### 2. Install Dependencies

Pastikan kamu sudah menginstall Node.js. Jalankan perintah berikut untuk menginstall semua library yang dibutuhkan:

```bash
npm install

```

### 3. Jalankan Development Server

```bash
npm run dev

```

Setelah dijalankan, buka `http://localhost:5173` di browser kamu.
